## Stickers

Drop replacement sticker images (PNG recommended) into this folder.

The UI references stickers via simple paths like:
- `/stickers/star.png`
- `/stickers/tape-blue.png`

### Star variants (4 PNGs)

To use your 4 star PNGs, add these files:
- `star-1.png`
- `star-2.png`
- `star-3.png`
- `star-4.png`

No code changes needed after that.

To swap stickers:
- Replace the files with your own PNGs **using the same filenames**, or
- Update `src/lib/stickers.ts` to point to new filenames.
