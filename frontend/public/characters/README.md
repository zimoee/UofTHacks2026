## Character images

To replace the temporary bunny art used on `/character-profile`:

1) Save your drawing as a PNG named `bunny.png`
2) Put it in this folder: `frontend/public/characters/bunny.png`

The app will automatically use it (with a fallback to `bunny-pixel.svg` if missing).

## Home header mascots

To replace the 3 mascots in the top bar (home page only), add these PNGs:

- `bunny.png`
- `turtle.png`
- `cat.png`

Put them here: `frontend/public/characters/`

If any PNG is missing, the app falls back to:
- `bunny-pixel.svg`
- `turtle-pixel.svg`
- `cat-pixel.svg`
